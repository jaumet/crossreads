<p>9 January, Thursday</p>
<p>F<br/>Go down town but get the cold shoulder from all so I return Home straight away &amp; get by mess time to talk until bed-time. Have long argument about Burford system as compared with this system of running a Hospital.</p>
<p>10 January, Friday</p>
<p>F<br/>S/S Lilley &amp; I go down town to YMCA. Band concert. We wait an Hour for the Pictures &amp; then return Home without seeing any. It is very "cold &amp; windy".</p>
